# Infercna

The goal of Infercna is to Infer copy number variation in single cell
data.

## Installation

You can install the development version of Infercna from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("Moonerss/Infercna")
```
