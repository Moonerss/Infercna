# scRNA-seq data from a patient with Glioblastoma

A single-cell RNA-sequencing dataset containing 8556 HQ genes and 1265
high quality cells. The data was generated using the Smart-Seq2 protocol
and was analysed in the form TPM.

## Usage

``` r
bt771
```

## Format

A matrix with 8556 genes (rows) and 1265 cells (columns):
